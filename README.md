# SpringBeginnerAnswer

## Description
こちらは、Trainocateで提供しているSpring Bootの研修用動画教材の解答プロジェクトになります。
以下のようなサイトで販売しています。
https://www.udemy.com/course/java_spring_beginner/

扱っているのはSpringフレームワーク、Thymeleaf、Gradleなどです。
こちらは確認ページ付きの問い合わせフォームの完成版プロジェクトとなっております。

## Requirement
Spring Bootを教材中ではSTSを使って動作させています。
STSはSpring用のEclipseです。

## Usage
ダウンロード、もしくはgitでcloneして試してください。
実行の仕方は教材中で説明されています。
